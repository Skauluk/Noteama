Hackathon 2019

Участие нашей команды в конкурсе Hackathon 2019 заключается в создании Front-End сайта, который является сайтом-визиткой нашей команды.

О нас

Наша команда называется No Team, которая занимается Front-End разработкой для конкурса Hackhaton 2019. Разработанный сайт является визиткой для продвижения нашей команды. Кманда состоит из 3 человек, которые учатся в одной группе 3 курса, у которых за плечами знания html, C#, Unity 3D и многие другие.

Ярослав

19 лет, студент 3 курса ГБ ПОУ ТУ ЭТТ группы ПКС-1-16. Является дизайнером сайта, владеет навыками работы в графическом редакторе, на базовом уровне html и css. Увлекается музыкой, рисованием и занимается системным администрированием.

Константин

18 лет, студент 3 курса ГБ ПОУ ТУ ЭТТ группы ПКС-1-16. Занимается созданием анимации и откликом сайта на действия, владеет навыками работы в html, css и JavaScript. Занимает должность гангера в различных онлайн мероприятиях.

Павел

20 лет, студент 3 курса ГБ ПОУ ТУ ЭТТ группы ПКС-1-16. Занимается вёрсткой сайта, владеет навыками работы в html и css. Увлекается музыкой и организовывает небольшие мероприятия.


Команда No Team предоставляет видео о техникуме ГБ ПОУ ТУ ЭТТ для ознакомления.
